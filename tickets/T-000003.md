## Goal
Write a one-shot Python script that runs pandapower shortcircuit for every (line, fault_type) combination and dumps bus voltages + line currents (pre-fault and at-fault) to chronogrid-demo/public/physics.json (~10KB).

## Acceptance Criteria
- [ ] `scripts/pandapower-physics/precompute.py` runs end-to-end without errors
- [ ] Output `chronogrid-demo/public/physics.json` exists and validates against the schema below
- [ ] Includes `pre_fault` block (steady-state) and `faults[<line_id>][<fault_type>]` blocks (one per scenario)
- [ ] Each fault scenario contains: `bus_voltages` (9 entries, pu), `line_currents` (6 entries, kA), `fault_current_ka`, `affected_lines` (overcurrent ≥1.5× rated)
- [ ] File size <50KB (sanity check that it's lookup data, not full simulation traces)

## Notes
Pandapower fault-type mapping (only LL/LG/3ph supported natively; phase-to-ground variants degenerate to same calc):
- `AG`/`BG`/`CG` → `fault="1ph"` with phase rotated
- `AB`/`AC`/`BC` → `fault="2ph"`
- `ABG`/`ACG`/`BCG` → `fault="2ph_g"` (line-to-line-to-ground)
- `ABCG` → `fault="3ph"`
- `NF` → no shortcircuit, return pre_fault as result

Schema (target):
```json
{
  "pre_fault": {
    "bus_voltages": {"1": 1.04, "2": 1.025, ...},
    "line_currents": {"L4-5": 0.85, "L4-6": 0.62, ...}
  },
  "faults": {
    "L4-5": {
      "AG": {"bus_voltages": {...}, "line_currents": {...}, "fault_current_ka": 12.4, "affected_lines": ["L4-5", "L5-7"]},
      "AB": {...},
      ...
    },
    ...
  }
}
```
