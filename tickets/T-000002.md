## Goal
Install pandapower in a local Python env and define the WSCC 9-bus network topology with line/transformer/generator/load parameters from the IEEE benchmark.

## Acceptance Criteria
- [ ] `scripts/pandapower_env/` directory exists with isolated `uv` venv (does NOT touch Render's runtime deps)
- [ ] `scripts/build_9bus_network.py` returns a `pandapowerNet` matching WSCC 9-bus benchmark (3 gens, 9 buses, 6 lines, 3 xfmrs, 3 loads)
- [ ] Running `pp.runpp(net)` converges with bus voltages within ±0.05 pu of published WSCC values
- [ ] Bus indices in code match diagram convention (buses 1/2/3 = generators, 5/6/8 = loads, 4/7/9 = step-up sides)

## Notes
Reference values for WSCC 9-bus (from IEEE benchmark):
- Bus 5 voltage: ~0.996 pu, Bus 6: ~1.013 pu, Bus 8: ~1.016 pu
- Total generation: ~319 MW, total load: ~315 MW

Keep the Python env at the repo root or in `scripts/` — must not pollute `requirements.txt` (Render does not need pandapower).
